## SWE 5186 이진수2

-![No Image](/assets/posts/20190806/1.png)

```python
for rounds in range(int(input())):
    num = float(input())
    result = ''
    cnt = 0
    while True:
        next_num = num*2
        print("next_num: ", next_num)
        result += str(int(next_num))
        print("result:", result)
        num = next_num-int(next_num)
        print("num: ", num)
        cnt += 1
        if num == 0.0:
            break
        if cnt > 13:
            break
    if cnt > 13:
        print(f"#{rounds+1} overflow")
    else:
        print(f"#{rounds+1} {result}")

```
