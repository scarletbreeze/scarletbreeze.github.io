## 1. 별 찍기

```python
T = int(input())
for i in range(1, T+1):
    print("*"*i)
```

## 2. 별 찍기

- i가 0일 때는 공백없이 출력되어야 함

```python
T = int(input())
for i in range(1, T):
    print(" " * (T-1-i), "*"*i)
print("*"*T)

```

## 3. 별 찍기

```python
T = int(input())
for i in range(T, 0, -1):
    print("*"*i)
```

## 4. 별 찍기

```python
T = int(input())
print("*"*T)
for i in range(T-1):
    print(" "*i, "*"*(T-i-1))
```

## 5. 별찍기

- 양 옆으로 칸 벌리는게 아니라, 올ㄴ쪽에는 공백이 없어.
- 예제 출력을 긁어보면 알 수 있다.

```python
T = int(input())
for i in range(T):
    print(" "*(T-i-1) + "*"*(2*i+1))
```

## 6. 별 찍기

```python
T = int(input())

```

## 7. 별 찍기

```python
T = int(input())
for i in range(T):
    print(" "*(T-i-1) + "*"*(2*i+1))
for i in range(T-2, -1, -1):
    print(" "*(T-i-1) + "*"*(2*i+1))
```

## 8. 별 찍기

```python
T = int(input())
for i in range(T-1, -1, -1):
    print("*"*(T-i) + " "*(2*i) + "*"*(T-i))
for i in range(1, T):
    print("*"*(T-i) + " "*(2*i) + "*"*(T-i))
```

## 9. 별 찍기

```python
T = int(input())

    print(" "*i + "*"*(2*(T-i)-1))

    print(" "*i + "*"*(2*(T-i)-1))
```
