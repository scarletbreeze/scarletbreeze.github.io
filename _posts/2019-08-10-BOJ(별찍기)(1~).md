## 1. 별 찍기

```python
T = int(input())
for i in range(1, T+1):
    print("*"*i)
```

## 2. 별 찍기

- i가 0일 때는 공백없이 출력되어야 함

```python
T = int(input())
for i in range(1, T):
    print(" " * (T-1-i), "*"*i)
print("*"*T)

```


## 5. 별찍기

- 양 옆으로 칸 벌리는게 아니라, 올ㄴ쪽에는 공백이 없어.
- 예제 출력을 긁어보면 알 수 있다.

```python
T = int(input())
for i in range(T):
    print(" "*(T-i-1) + "*"*(2*i+1))
```

## 15. 별 찍기

