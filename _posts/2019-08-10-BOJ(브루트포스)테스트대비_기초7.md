## 7. 모든 순열

- N과 M (1) 문제와 완전히 동일

```python
def DFS(s):
    global result
    if len(result) == T:
        print(" ".join(result))
        return
    for i in range(1, T+1):
        if str(i) not in result:
            result += str(i)
            DFS(i)
            result = result[:-1]
T = int(input())
result = ""
DFS(0)

```
